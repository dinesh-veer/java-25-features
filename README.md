# 📌 Project Overview

**java-25-features** is a practical learning repository demonstrating **new features introduced in Java 25**.

This project provides **clear, runnable examples** explaining modern Java capabilities and the concepts behind **JEPs (Java Enhancement Proposals)**.

The repository is designed for developers who want to:

* Learn Java 25 features quickly
* Explore practical implementations
* Understand new APIs and language improvements
* Experiment with modern Java capabilities

# 🧠 Java 25 JEP Overview

| JEP     | Feature                                      | Description                                                   |
| ------- | -------------------------------------------- | ------------------------------------------------------------- |
| JEP 506 | Scoped Values                                | Safer alternative to `ThreadLocal` for structured concurrency |
| JEP 511 | Module Import Declarations                   | Import modules directly in Java source files                  |
| JEP 512 | Compact Source Files & Instance Main Methods | Simplifies writing simple Java programs                       |
| JEP 513 | Flexible Constructor Bodies                  | Allows validation before calling `super()`                    |

These examples help developers understand **how JEPs translate into real code**.

# 🚀 Getting Started

## Requirements

* **Java 25**
* **Maven**
* Git

---

## Clone Repository

```bash
git clone https://github.com/dinesh-veer/java-25-features.git
cd java-25-features
```

---

## Build Project

```bash
mvn clean compile
```

---

## Run Examples

If preview features are used:

```bash
java --enable-preview ExampleClass
```
